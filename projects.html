Cloud-Based Appointment Management System

A production-ready healthcare appointment booking platform built with a modern AWS multi-tier architecture. Demonstrates secure, highly available, and cost-efficient infrastructure for patient scheduling and management. Emphasizes automation, monitoring, and operational resilience.

Architecture: S3 + CloudFront (HTTPS via Route 53) • API Gateway + Lambda (Node.js) • DynamoDB (conditional writes) • Cognito (JWT Auth) • CloudWatch (logs, alarms, dashboards) • X-Ray tracing • WAF + KMS encryption • IAM least privilege • GitHub Actions CI/CD • Terraform IaC

  • Atomic booking with DynamoDB conditional writes (no double-booking).
  • End-to-end encryption, authentication, and audit logging.
  • Budget alarms + auto-teardown for cost control.
  • Designed for zero idle cost and fast recovery from common failure scenarios.

Cloud Game Leaderboard

A fully serverless real-time leaderboard system showcasing scalable event-driven design. Players can submit scores and view rankings through a low-latency API. Designed for zero infrastructure management and optimal AWS free-tier usage.

Architecture: S3 (static frontend) • API Gateway (REST + WebSocket) • Lambda (Node.js) • DynamoDB (top-N leaderboard with GSI) • CloudWatch (monitoring & alarms) • WAF (rate limiting) • GitHub Actions + Terraform

  • Real-time leaderboard updates via WebSocket or REST polling.
  • Optimized GSI design for top-N queries without hot partitions.
  • WAF rules and throttling to prevent abuse and spam.
  • CI/CD one-click deploy and destroy with Terraform.

Serverless Contact Form Application

A lightweight, production-grade contact form application with spam protection, encryption, and zero-cost infrastructure. Ideal for personal sites and small businesses that require reliable submission handling without overhead.

Architecture: API Gateway → Lambda → DynamoDB (TTL cleanup) → S3 + SNS • CloudWatch • GitHub Actions (CI/CD)

  • Fully serverless with $0 idle cost.
  • Spam protection with honeypot and WAF rate limits.
  • Automated record cleanup using DynamoDB TTL.
  • Real-time notifications via SNS with CloudWatch metrics and alarms.
